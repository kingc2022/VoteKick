## 支持的版本
> Only Minecraft 1.20.1

## 权限节点

|      Name       | Default |
|:---------------:|:-------:|
| votekick.start  |  true   |
|  votekick.vote  |  true   |
| votekick.reload |   op    |
|  votekick.stop  |   op    |
| votekick.bypass |  false  |

## 命令

> /vote start <玩家名> 开启踢出玩家的投票
> 
> /vote stop 强制关闭当前投票 
> 
> /vote reload 重新加载配置文件 
>
> /vote 投票踢出玩家
